# Privacy Policy

## Overview
This Azure DevOps Marketplace extension does **not** collect, store, transmit, or process any personal data, customer content, or telemetry outside of the Azure DevOps environment.

## Data Collection
This extension:
- Does **not** collect personal or organizational data
- Does **not** transmit data to external services
- Does **not** store data outside the Azure DevOps tenant where it is installed

All functionality executes within the context of Azure DevOps pipelines or services using the permissions explicitly granted by the user.

## Authentication & Permissions
The extension operates using Azure DevOps–provided authentication and respects all security, identity, and access controls enforced by the Azure DevOps platform.

## Third‑Party Services
This extension does **not** integrate with or send data to any third‑party services.

## Changes to This Policy
If future versions of this extension introduce data collection or external connectivity, this privacy policy will be updated accordingly.

## Contact
If you have questions or concerns about this privacy policy, please open an issue in the repository.
